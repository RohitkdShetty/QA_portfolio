## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/RohitkdShetty/QA_portfolio.git
```

Navigate to the project directory:

```bash
cd QA_portfolio
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
